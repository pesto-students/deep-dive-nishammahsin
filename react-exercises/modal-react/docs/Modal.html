# <Modal title footer closable okText cancelText okButtonProps cancelButtonProps size width styles visible onClose />

Modal component

PropTypes:
Name Type Required Description Default
title union No The modal dialog's title
footer element No Footer content, ReactNode set as footer={null} when you don't need default buttons
closable bool No Whether a close (x) button is visible on top right of the modal dialog or not true
okText string No Text of the OK button. default 'OK' function()
cancelText string No Text of the Cancel Button. default 'CANCEL' function()
okButtonProps object No OK button props
cancelButtonProps object No Cancel button props
size enum No Size of the modal .small | medium | large default is medium function()
width union No width of the modal function()
styles object No styles of the modal {}
visible bool No Whether the modal dialog is visible or not false
onClose func No Specify a function that will be called when a user clicks mask, close button on top right or Cancel button
